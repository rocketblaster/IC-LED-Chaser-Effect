# LED Chaser Effect PCB

This is an LED chaser effect with a custom shaped PCB edge cut. It uses NE555 and CD4017 integrated circuits to achieve this effect.

![3D PCB image front](attachments/image24.png)
![3D PCB image back](attachments/image25.png)

How this will work is the LEDs will light up in a pattern and basically feel like it is chasing something bouncing back and forth.
